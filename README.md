# Power BI Dashboard Demonstration Project

## Overview
This repository contains a Power BI dashboard created as a demonstration project to showcase interactive data visualisation and dashboard design using Microsoft Power BI. The dashboard allows users to explore key metrics, trends, and category-based insights through interactive visuals and filters.

The project is suitable for portfolio/demo purposes, presentations, and learning Power BI reporting techniques.

---

## Key Features
- KPI Cards for quick summary metrics
- Trend analysis charts (line/area) to show changes over time
- Forecasting (optional) to highlight expected future patterns
- Geographic visual (map/filled map) for spatial insights
- Comparison visuals (bar/column charts)
- Interactive slicers/filters for custom exploration
- Clean layout and formatting following dashboard best practices

---

## Dashboard Interaction (How to Use)
1. Download and open the `.pbix` file using Microsoft Power BI Desktop.
2. Use slicers/filters to select different categories, regions, or time periods.
3. Hover over charts to view tooltips with detailed values.
4. Click on visuals to cross-filter other charts automatically.
5. Clear slicer selections to reset the dashboard.
